پریمیوم (ویژه)

مقدمه

ما همواره در تلاش بوده و هستیم تا گروهی آرام را برای تمامی همراهان مجموعه گروه بان بسازیم

دستورات گروه

سینتکس:
  • <> : آرگومان های نیازمند
  • [ ] : آرگومان های غیر لازم
  • ( ) : نام دیگر دستور

اصلی

  • ستاپ
    • توضیحات: کانفیگ ربات و تنظیم عملکرد ها
    Usage : /setup

    دسترسی ها: ادمین

  • پاسخگوی خودکار
    • توضیحات: دریافت راهنمای بخش پاسخگویی خودکار و لیست دستورات بخش
    Usage : /auto-answer (/auto_answer)

    دسترسی ها: ادمین

  • ضد کلمات
    • توضیحات: دریافت راهنمای بخش ضد کلمات و لیست دستورات
    Usage : /anti-word (/anti_word)

    دسترسی ها: ادمین

  • خوش آمد گو
    • توضیحات: دریافت راهنمای بخش خوش آمد گوی گروه و لیست دستورات
    Usage : /w

    دسترسی ها: ادمین

فرعی

  • اطلاعات گروه
    • توضیحات: دریافت اطلاعات کلی و ادمینان گروه
    Usage : /groupinfo

    سپر امنیتی گروه بان در گروه
